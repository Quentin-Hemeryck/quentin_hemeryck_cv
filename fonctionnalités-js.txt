Fonctionnalités JavaScript intégrées dans le site :

• Bouton d'impression : permet d'imprimer le CV via un clic sur une icône.
• Particules d’arrière-plan animées grâce à la bibliothèque particles.js.
• Initialisation automatique de l'animation au chargement du DOM.
• Message de confirmation dans la console ("Script CV chargé avec succès").
